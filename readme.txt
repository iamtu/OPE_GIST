GIST package version 1.0 (March, 2013)

Before you run the codes, please first run "install.m" to install the package


You are suggested to read the manual (available at the manual folder) before using the functions.


The folder "GIST" includes all functions implemented in this package

The folder "Examples" includes some examples to show how to use this package

The folder "data" includes a data set used in examples

The folder "manual" includes a manual on how to use this package.


If you use this package, please cite:

P. Gong, C. Zhang, Z. Lu, J. Huang, and J. Ye. GIST: General Iterative
Shrinkage and Thresholding for Non-convex Sparse Learning.
Tsinghua University, 2013. http://www.public.asu.edu/~jye02/Software/GIST.

P. Gong, C. Zhang, Z. Lu, J. Huang, and J. Ye. 
A General Iterative Shrinkage and Thresholding Algorithm for Non-convex
Regularized Optimization Problems. ICML 2013.
